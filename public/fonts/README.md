# Base Neue Font Files

This directory should contain the Base Neue font files in the following formats:

## Required Font Files:

### Regular Weight (400)
- `base-neue-regular.woff2`
- `base-neue-regular.woff`

### Medium Weight (500)
- `base-neue-medium.woff2`
- `base-neue-medium.woff`

### Semibold Weight (600)
- `base-neue-semibold.woff2`
- `base-neue-semibold.woff`

### Bold Weight (700)
- `base-neue-bold.woff2`
- `base-neue-bold.woff`

### Monospace Variant
- `base-neue-mono-regular.woff2`
- `base-neue-mono-regular.woff`

## Font License

Base Neue is a commercial font family. Please ensure you have the proper license before using these fonts in production.

## Fallback Fonts

The system is configured with proper fallback fonts (Inter, system-ui, sans-serif) so the application will work even without the Base Neue font files.
